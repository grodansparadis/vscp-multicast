# VSCP multicast excample for Wiznet W55RP20-EVB-Pico

## Setup

First set the board name in CMakeLists.txt

set(BOARD_NAME W5500_EVB_PICO)

Check [here](https://github.com/WIZnet-ioNIC/WIZnet-PICO-C?tab=readme-ov-file#ethernet-example-testing) for the correct board name
